HTML is like the skeleton of a webpage. It tells the computer what parts go where!

CSS is like the clothes and colors for the webpage. It makes everything look nice and pretty.

JavaScript is the magic that makes webpages do cool things, like buttons you can click!

Let's make your very first webpage together! Follow these easy steps:

Step 1: Start with the basic structure

<!DOCTYPE html>
<html>
<head>
  <title>My First Webpage</title>
</head>
<body>

This tells the browser you are writing an HTML page and sets the title.

Step 2: Add a heading

  <h1>Welcome to My Page!</h1>

This heading will show big and bold on your page.

Step 3: Add a paragraph

  <p>This is my very first webpage, and I made it myself!</p>

This paragraph shows some text below the heading.

Step 4: Close the body and html tags

</body>
</html>

These tags tell the browser the page content is finished.

Try copying all this code into the editor and click Run!
